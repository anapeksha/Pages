## Getting Started

Instructions on how to set up the project.

### Prerequisites

[Node.js](https://nodejs.org) v16.15.0.

```sh
corepack enable
```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/anapeksha/Pages.git
   ```
2. Install NPM packages

   ```sh
   yarn install
   ```

3. Start the Development server

   ```sh
   yarn start
   ```

4. Create a production build
   ```sh
   yarn build
   ```
